Noor Al-Qalb

A full-stack web application that uses a Hugging Face ML model API to predict a user's emotion from their text input. It then dynamically displays a corresponding Quranic verse with translation and audio playback, offering a unique blend of technology and spiritual support.

Technologies Used:

Python Hugging Face API Flask HTML5 CSS3 JavaScript Bootstrap
