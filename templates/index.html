Tom and Jerry Emotion Detection ANN Model

The Tom and Jerry Emotion Detection Project is an AI-based application built using Artificial Neural Networks (ANNs) and PyTorch. The system was designed to detect and classify emotions from cropped face images of the Tom and Jerry cartoon characters.

The workflow began with creating CSV datasets by extracting bounding box coordinates from XML annotations, followed by applying image augmentations such as horizontal and vertical flips to enhance dataset diversity. Cropped face images were then generated and mapped with corresponding emotion labels for training.

A pretrained MobileNetV2 model was fine-tuned on this dataset to classify five different emotion categories. The model was trained using PyTorch with a custom dataset loader, data transformations, and optimization setup. After achieving accurate results, the trained model was saved as a .pth file.

To make the project interactive, a web application was also developed where users can upload cropped character face images and receive real-time emotion predictions powered by the trained model.

The complete code and files are available on GitHub, demonstrating skills in computer vision, deep learning, dataset preparation, transfer learning, and web app integration.
